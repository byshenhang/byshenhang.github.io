如何手动安装SideFX Labs

2023-05-28 /
  • Houdini
 / 点击 / 阅读耗时 1 分钟

介绍

部分Houdini版本自动安装SideFx Lab会提示找不到文件，这里简单介绍下如何手动安装SideFx Lab

前提

从 github.com/sideeffects/SideFXLabs 上Clone或直接下载其中的内容。选择一个合适的位置将内容放里面：

然后，将其中的SideFXLabs.json文件拷贝到$HOME/Houdini18.5/packages文件夹（与本机Houdini版本对应，如果没有packages文件夹则新建一个）:

最后，修改SideFXLabs.json的内容，将”path”的值改为你的 SideFXLabs 的位置。

打开Houdini，即可看到出现工具了：
